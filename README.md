# 🎬 Cinema Seat Booking System (C++)

A console-based application developed in C++ to simulate a cinema seat booking system using a 2D array.

## Features
- Menu-driven interface for booking, cancellation, and viewing seats  
- Displays seat layout with available and booked seats  
- Shows total booked and available seats  
- Prevents over-booking and invalid cancellations  
- Displays "Housefull" when no seats are available  

## Technologies Used
- C++ (Arrays, Functions, Loops, Switch Statements)

## How It Works
- Seats are stored in a 2D array  
- `0` represents available seats and `1` represents booked seats  
- Users interact through a menu to perform actions  

## How to Run
1. Compile:
   g++ filename.cpp -o output  
2. Run:
   ./output
